# Optional Authenticated Development Connectors

This folder documents optional workflows for users who want to use their own authenticated AI coding environment.

The workbench does not include shared keys, OAuth secrets, service accounts, or a hosted backend. Users authenticate directly with their own provider tools.

Supported workflow targets:

- Codex/OpenAI-style authenticated coding environments.
- Google Antigravity-style authenticated coding environments.

Connector files in this folder should remain documentation or local adapters only. They must never hardcode secrets.
