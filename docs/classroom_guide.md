# Classroom Guide

## Exercise 1: Resonance Sanity Check

Calculate an LC resonant frequency and compare it to the output of `core.resonance.lc_resonance_hz`.

## Exercise 2: Evidence Labels

Take one claim and decide whether it is:

- `confirmed`
- `modeled`
- `hypothesis`
- `unsupported`

Then attach one URL from `docs/source_ledger.md`, unless the claim is explicitly `hypothesis` or `unsupported`.

## Exercise 3: Fractal Wave-Friction Reading

Create a bounded system, scale interval, carrier, loss ratio, and boundary instability. Explain why `dF` is not the same as generic uncertainty.
