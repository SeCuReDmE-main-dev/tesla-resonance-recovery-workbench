# SecuredMe Education Visual Theme

This folder contains the shared SecuredMe Education visual identity for the open-source school-tool suite.

## Assets

- `securedme-education-logo-light.png`: light logo sheet.
- `securedme-education-logo-dark.png`: dark logo sheet.
- `securedme-education-banner-light-thin.png`: light thin banner.
- `securedme-education-banner-dark-thin.png`: dark thin banner.

## Palette

- Deep Navy: `#161B6A`
- Electric Blue: `#23B8FF`
- Vivid Violet: `#6F42FF`
- Light Gray: `#D9DDE5`
- White: `#FFFFFF`

## Boundary

The Education theme is the default public visual identity for SecuredMe open-source school tools. These assets are copied without compression or transformation. This pass does not create tool-specific logos, does not change the repository license, and does not mark any repository as stable. Each repository remains pre-alpha / in development unless its main README states otherwise.
