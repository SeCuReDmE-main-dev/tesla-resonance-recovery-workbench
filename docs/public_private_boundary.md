# Public / Private Boundary

## Public Repository

`tesla-resonance-recovery-workbench`

Allowed:

- source ledgers;
- reusable math modules;
- validation payloads;
- education docs;
- connector workflow docs;
- safe civic/scientific examples.

Forbidden:

- raw private email;
- unpublished blog drafts;
- private Dr. Aguilar context;
- credentials;
- retaliation or targeting instructions.

## Private Repository

`tesla-resonance-blog-private`

Allowed:

- narrative drafts;
- three-voice whitepaper drafts;
- 1910 Tesla Council scenes;
- private civic context;
- redaction staging.

Only sanitized files from `publication_ready/` can be considered for public release.
