# Teacher Guide

This project can support lessons on:

- primary-source reading;
- Tesla's resonance vocabulary;
- mathematical modeling limits;
- evidence labels;
- fractal/neutrosophic uncertainty;
- ethical AI-assisted research.

Suggested classroom boundary:

- Run local validation first.
- Treat AI coding accounts as optional accelerators.
- Require students to label claims as `confirmed`, `modeled`, `hypothesis`, or `unsupported`.
- Keep private political/contextual drafts outside the public tool repository.
