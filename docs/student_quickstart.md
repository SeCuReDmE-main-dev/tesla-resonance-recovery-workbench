# Student Quickstart

You can use the workbench without a paid AI account.

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
python -m pytest
python -m validation.fnp_qnn_payload --out output\fnp_qnn_payloads
```

Use an authenticated Codex/OpenAI or Google Antigravity account only if your school, teacher, or personal plan allows it.

Do not upload private classroom, personal, or political material unless your teacher explicitly tells you it is allowed.
